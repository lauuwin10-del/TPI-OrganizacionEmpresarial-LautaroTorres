# TPI-OrganizacionEmpresarial-LautaroTorres

## 📌 Descripción
Este repositorio contiene el **Trabajo Práctico Integrador (TPI)** de la materia **Organización Empresarial** de la UTN-TUPaD.  

El proyecto implementa un **chatbot en Python** para la **Gestión de Vacaciones**, basado en un modelo de procesos BPMN 2.0.  
El sistema permite a los empleados:
- Consultar sus días de vacaciones disponibles.
- Solicitar períodos de descanso.
- Validar legajo, fechas y cantidad de días solicitados.
- Simular la aprobación o rechazo por parte de RRHH.  

Además, se incluyen:
- **Diccionario de datos** con las tablas de empleados y solicitudes.  
- **Manual de usuario** con ejemplos de interacción en consola.  
- **Pruebas de robustez** que muestran cómo el sistema maneja errores de entrada y caminos alternativos.  
- **Informe en PDF** con el desarrollo completo del trabajo.  
- **Diagrama BPMN** que representa el flujo del proceso administrativo.  

Este proyecto integra conocimientos de programación y organización empresarial, demostrando cómo la tecnología puede optimizar procesos de negocio y preparar al estudiante para desempeñarse como **consultor tecnológico**.
